# Chromium GN string_split 함수 가이드

`string_split` 함수는 입력 문자열을 지정된 구분자(Separator)를 기준으로 나누어 문자열 리스트로 반환합니다. `string_join`과 반대되는 기능을 수행하며, 공백이나 쉼표로 구분된 문자열을 파싱할 때 유용합니다.

## 1. 기본 문법 (Basic Syntax)

```gn
result = string_split(input_string, separator)
```

*   **`input_string`**: 분할할 원본 문자열입니다.
*   **`separator`**: 문자열을 나눌 기준이 되는 구분자 문자열입니다.
*   **반환값**: 분할된 문자열들의 리스트입니다.

## 2. 작동 방식 (How it works)

`string_split`는 `input_string`에서 `separator`를 찾아, 그 위치를 기준으로 문자열을 나눕니다.

```gn
my_string = "a,b,c"

# ","를 기준으로 분할
result = string_split(my_string, ",")

# result는 [ "a", "b", "c" ]가 됩니다.
print(result)
```

## 3. 사용 예시 (Usage Examples)

### 3.1. 경로 분해

파일 경로를 각 디렉토리 요소로 분해할 때 사용할 수 있습니다.

```gn
path = "path/to/my/file"
path_parts = string_split(path, "/")

# path_parts는 [ "path", "to", "my", "file" ]이 됩니다.
```

### 3.2. 공백으로 구분된 플래그 파싱

하나의 문자열로 정의된 컴파일러 플래그들을 개별 플래그 리스트로 변환합니다.

```gn
flags_str = "-Wall -Werror -O2"
flags_list = string_split(flags_str, " ")

# flags_list는 [ "-Wall", "-Werror", "-O2" ]가 됩니다.
```

## 4. 주의사항

*   **빈 문자열**: 구분자가 연속으로 나오거나 문자열의 시작/끝에 있으면, 결과 리스트에 빈 문자열(`""`)이 포함될 수 있습니다.
    ```gn
    string_split("a,,b", ",")  # 결과: [ "a", "", "b" ]
    ```
*   **구분자 없음**: 입력 문자열에 구분자가 없으면, 원본 문자열 하나만 포함하는 리스트가 반환됩니다.
