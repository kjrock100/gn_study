# Chromium GN string_join 함수 가이드

`string_join` 함수는 구분자(Separator)와 문자열 리스트를 입력받아, 리스트의 모든 요소를 구분자로 연결한 하나의 문자열을 반환합니다. 컴파일러 플래그를 구성하거나, 스크립트 인자로 전달할 리스트 문자열을 만들 때 유용합니다.

## 1. 기본 문법 (Basic Syntax)

```gn
result = string_join(separator, list)
```

*   **`separator`**: 리스트 요소 사이에 삽입할 구분자 문자열입니다. (예: `", "`, `" "`, `""`)
*   **`list`**: 연결할 문자열들의 리스트입니다.
*   **반환값**: 연결된 단일 문자열입니다.

## 2. 사용 예시 (Usage Examples)

### 2.1. 쉼표로 구분된 목록 만들기

스크립트 인자나 설정 파일에 쉼표로 구분된 값을 전달해야 할 때 사용합니다.

```gn
my_list = [ "a", "b", "c" ]
joined = string_join(", ", my_list)

# joined는 "a, b, c"가 됩니다.
print(joined)
```

### 2.2. 경로 연결하기

여러 경로 요소를 하나의 경로 문자열로 합칠 때 사용할 수 있습니다.

```gn
path_parts = [ "usr", "local", "bin" ]
full_path = string_join("/", path_parts)

# full_path는 "usr/local/bin"이 됩니다.
```

### 2.3. 컴파일러 플래그 구성

리스트에 있는 플래그들을 공백으로 연결하여 하나의 문자열로 만들 때 유용합니다.

```gn
flags = [ "-Wall", "-Werror", "-O2" ]
flags_str = string_join(" ", flags)

# flags_str는 "-Wall -Werror -O2"가 됩니다.
```

## 3. 주의사항

*   **문자열 리스트**: `list` 인자의 모든 요소는 반드시 문자열이어야 합니다. 정수나 불리언이 포함되어 있으면 에러가 발생할 수 있습니다.
*   **빈 리스트**: 입력 리스트가 비어있으면 빈 문자열(`""`)을 반환합니다.
*   **단일 요소**: 리스트에 요소가 하나만 있으면, 구분자 없이 해당 요소만 반환됩니다.
