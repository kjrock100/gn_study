# Chromium GN string_replace 함수 가이드

`string_replace` 함수는 입력 문자열에서 특정 부분 문자열을 찾아 다른 문자열로 치환(Replace)하는 데 사용됩니다. 파일 경로를 조작하거나, 문자열 템플릿을 채울 때 유용합니다.

## 1. 기본 문법 (Basic Syntax)

```gn
result = string_replace(input_string, old_substring, new_substring)
```

*   **`input_string`**: 변환할 원본 문자열입니다.
*   **`old_substring`**: 찾아서 바꿀 부분 문자열입니다.
*   **`new_substring`**: `old_substring`을 대체할 새로운 문자열입니다.
*   **반환값**: 치환이 완료된 새로운 문자열입니다.

## 2. 작동 방식 (How it works)

`string_replace`는 `input_string` 내에서 `old_substring`과 일치하는 **모든** 부분을 `new_substring`으로 바꿉니다.

```gn
my_string = "hello world, hello gn"

# "hello"를 "goodbye"로 치환
result = string_replace(my_string, "hello", "goodbye")

# result는 "goodbye world, goodbye gn"이 됩니다.
print(result)
```

## 3. 사용 예시 (Usage Examples)

### 3.1. 파일 확장자 변경

```gn
source_file = "my_image.jpg"
output_file = string_replace(source_file, ".jpg", ".png")

# output_file은 "my_image.png"가 됩니다.
```

### 3.2. 경로 구분자 변경 (Windows)

GN은 내부적으로 `/`를 사용하지만, 외부 스크립트에 Windows 스타일 경로를 전달해야 할 때 사용할 수 있습니다.

```gn
unix_path = "path/to/file"
win_path = string_replace(unix_path, "/", "\\")

# win_path는 "path\\to\\file"이 됩니다.
```

## 4. 주의사항

*   **리스트 처리 불가**: 이 함수는 단일 문자열에 대해서만 동작합니다. 리스트의 각 요소에 대해 치환을 수행하려면 `foreach` 반복문을 사용해야 합니다.
*   **정규식 미지원**: `old_substring`은 정규식(Regular Expression)이 아닌, 고정된 문자열입니다. 복잡한 패턴 매칭은 지원하지 않습니다.
