# Chromium GN Help 명령어 가이드

`gn help`는 GN 빌드 시스템의 내장 문서(Built-in Documentation)를 조회하는 명령어입니다. GN은 별도의 웹 문서를 찾아보지 않아도, 터미널에서 거의 모든 기능에 대한 상세한 설명을 확인할 수 있도록 설계되었습니다.

## 1. 기본 사용법 (Basic Usage)

```bash
gn help
```

이 명령어를 실행하면 사용 가능한 모든 명령어(Commands)와 도움말 토픽(Topics)의 목록을 보여줍니다.

## 2. 특정 명령어 도움말

특정 명령어가 어떻게 동작하는지, 어떤 옵션을 사용할 수 있는지 알고 싶다면 명령어 이름을 인자로 전달합니다.

```bash
gn help gen
gn help args
gn help desc
```

## 3. 변수 및 함수 도움말

`BUILD.gn` 파일에서 사용하는 내장 변수나 함수에 대한 설명도 조회할 수 있습니다.

### 변수 (Variables)
```bash
gn help sources
gn help cflags
gn help target_cpu
```

### 함수 (Functions)
```bash
gn help executable
gn help template
gn help import
```

## 4. 유용한 도움말 토픽 (Topics)

단순한 명령어 설명 외에도, GN 언어의 문법이나 동작 원리에 대한 심층적인 가이드도 제공합니다.

*   **`gn help grammar`**: GN 언어의 문법과 구문 구조에 대한 설명.
*   **`gn help execution`**: 빌드 파일이 로드되고 실행되는 과정 설명.
*   **`gn help dotfile`**: `.gn` 파일(루트 설정 파일)의 역할과 구조 설명.
*   **`gn help input_conversion`**: `exec_script` 등의 입력 변환 모드 설명.
*   **`gn help runtime_deps`**: 런타임 의존성 관리 방법 설명.

## 5. 스위치 (Switches)

GN 명령어 전반에 걸쳐 사용할 수 있는 커맨드 라인 플래그(Switch)에 대한 정보입니다.

```bash
gn help switches
```

*   `--root`: 소스 루트 디렉토리 지정.
*   `--dotfile`: 사용할 `.gn` 파일 경로 지정.
*   `--color`: 출력 색상 제어.
*   `--time`: GN 실행 시간 측정.
